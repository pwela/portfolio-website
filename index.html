Event application (meet App)

This app will allow users to search for a city and get a list of events hosted in that city. It also displays two charts, one that shows how many events will take place in each location (via a scatterplot), and another that visualizes the popularity of event genres (via a pie chart). This is a serverless progressive web application (PWA) built with React, AWS Lambda and Google calendar API. The test-driven development approach (TDD) was also followed during the building process using Jest, Cumcumber and Puppeter for testing.
